Markdown all in one(Plug)

  • 强调：

    • Ctrl+B–>粗体。
    • Ctrl+I–>斜体。

  • 列表：

    • 1.+Enter直接换为2.；
    • Enter+Tab切为子标题。

  • 超链接：选中文字+ctrl+v。

  • 插入图片：复制图片+ctrl+alt+v

  • 标记：`文字`

    • eg

  • 区块引用：一级嵌套：> /二级嵌套：>>

    Outside

    Inside

  • 代码区块： ```+代码高亮标签+Enter+```

echo<<"Hello World";
  • 反斜杠：\
    相当于反转义作用。使符号成为普通符号。
